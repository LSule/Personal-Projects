This code was created by watching tutorial by Kenny Yip Coding https://www.youtube.com/@KennyYipCoding

Most I can claim is the patch to two ace starter hand bug, adding enter function to place bets, and adaptable labels for the stay button.

must have java version to run.

To run.
Open a terminal to /scr.
type and execute javac app.java
type and execute javac blackjack.java
type and execute java app.java

That should open a window with game.
